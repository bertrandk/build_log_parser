# build_log_parser

Fetch metrics from build logs: tests, duration, coverage, etc.

## Installation

Install via rubygems:

```
gem install build_log_parser
```

Or with bundler:

```
gem "build_log_parser"
```

## Tests

Execute test suite by running the following command:

```
bundle exec rake test
```
